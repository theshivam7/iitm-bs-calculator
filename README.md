# IITM BS Calculator

![GitHub License](https://img.shields.io/badge/license-MIT-blue.svg)

## Introduction

Welcome to the IITM BS Calculator, a Python program that calculates the minimum marks you need in the final exam to pass a course. This tool was created by Shivam Sharma to help students of IITM BS degree.
## Features

- Calculate the minimum marks needed in the final exam to pass a course.
- Account for the weighted contribution of quizzes to the final grade.
- Easy-to-use command-line interface.

## Getting Started

1. Clone this repository to your local machine:

   ```shell
   git clone https://github.com/yourusername/iitm-bs-calculator.git
   ```

2. Navigate to the project directory:

   ```shell
   cd iitm-bs-calculator
   ```

3. Run the program:

   ```shell
   python iitm_bs_calculator.py
   ```

4. Follow the on-screen instructions to input your information and quiz scores.

5. The program will calculate and display the minimum marks needed to pass the course.

## Usage

Here's how to use the IITM BS Calculator:

1. When prompted, enter your name.

2. Input your quiz scores as percentages for Quiz 1 and Quiz 2.

3. The program will calculate the weighted scores for each quiz and determine which quiz contributes to your grade.

4. It will then calculate the minimum marks required in the final exam to pass the course.

5. The result will be displayed on the screen.
